## Install

```bash
# use pnpm
$ pnpm install -D @node-kit/extra.fs

# use yarn
$ yarn add -D @node-kit/extra.fs

# use npm
$ npm install -D @node-kit/extra.fs
```

## Usage

```js
import { functionName } from '@node-kit/extra.fs'

// run functionName
functionName()
```
